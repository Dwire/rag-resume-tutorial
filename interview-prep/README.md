# Interview prep bank

Your growing, searchable interview revision deck. **One file per syllabus step**,
named `NN-<topic>.md` (e.g. `02-chunking.md`, `07-grounding.md`).

## What goes here

At the end of each step, the step's **Key Takeaways** and its **interview Q&A with
model answers** get appended to that step's file — so the questions and answers
live on disk, not just in a chat transcript you'll lose. If you're working solo
(no AI tutor), write them yourself as you finish each step.

Suggested per-file shape:

```markdown
# Step NN — <topic>

## Key Takeaways
- ...

## Q&A
**Q: <question an interviewer might ask>**
A: <the answer, phrased the way you'd actually say it>
```

## The capstone

At **Step 12** these per-step files are compiled into a single
`INTERVIEW_CHEATSHEET.md` at the repo root — architecture recall, per-stage
trade-offs, and the top questions in one place. This folder is the raw material;
the cheat sheet is the distilled version.
